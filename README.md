# 大脚骑行 (bigbike)
大脚骑行是一款拥有专利技术的骑行软件，分为GPS与智能硬件两种模式，使用智能硬件模式时，需使用大脚骑行的配套智能硬件“音频码表”，因使用智能硬件测速，大脚骑行的骑行数据比“行者”等使用手机GPS测速的软件准确30%，能媲美高级专业码表。

## 发明专利 (patent)
2015获得中国专利局颁发的实用新型专利证书，欢迎极客拿去实验，请勿用作商业用途
